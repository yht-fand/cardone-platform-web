# 部门 - 查询列表

## 权限

> **read,usercenter-read,usercenter-department-read,usercenter-department-r0002-read**

> **read-my-department,usercenter-read-my-department,usercenter-department-read-my-department,usercenter-department-r0002-read-my-department**

> **read-my,usercenter-read-my,usercenter-department-read-my,usercenter-department-r0002-read-my**

## 路径

[./web-api/v1/usercenter/department/r0002.json](./r0002.json)

## 类型

HTTP GET

```
{
  "beginDate": "test-data",
  "createdByCode": "test-data",

  "endDate": "test-data",
  "lastModifiedByCode": "test-data",
  "lastModifiedDate": "test-data",
  "name": "test-data",
  "orgCode": "test-data",
  "parentCode": "test-data",
  "parentTreeCode": "test-data",

  "permissionCodes": "test-data",
  "roleCodes": "test-data",
  "stateCode": "test-data",
  "version": "test-data",
  "wfId": "test-data",
  "token": "test"
}
```

字段名|可为空|类型及范围|说明
